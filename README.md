# Аварийный электрик — Тайлер Шнурден

Одностраничный лендинг для услуги аварийного электрика. Выполнен в индустриальном, брутальном стиле.

## 🔧 Стек технологий
- HTML5, CSS3, JavaScript (Vanilla)
- BEM-нейминг
- CSS-переменные и переменные шрифты (`Inter`, `Source Code Pro`)
- Адаптивная верстка (Mobile First / Desktop)
- Кастомный слайдер и модальное окно без внешних библиотек

## 📷 Особенности
- Фиксированный хедер с плавным схлопыванием
- Бесконечная анимация предупреждающей полосы
- Кастомный чекбокс с анимацией встряхивания
- Модальное окно заявки с валидацией
- Адаптивный слайдер отзывов

## 📂 Запуск
Просто откройте `index.html` в любом браузере. Никаких сборщиков или npm не требуется.
